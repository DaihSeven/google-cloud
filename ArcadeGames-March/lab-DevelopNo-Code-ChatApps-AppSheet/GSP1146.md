💬 No-Code Chat Apps com AppSheet (Resumo)
📌 Visão geral

Neste lab você cria um app de chat sem código integrado ao Google Chat usando o AppSheet.

🎯 Caso prático:

Gerenciar manutenção de caixas eletrônicos (ATM)
Técnicos interagem direto pelo chat
🧠 Conceito principal

👉 No-code app = aplicação criada sem programação
👉 Chat app = app que roda dentro do Google Chat

💡 Benefício:

Não precisa abrir outro sistema
Tudo feito via chat (comandos, botões, automações)
🧩 Etapas do Lab
1. Criar o app
Copiar template ATM Maintenance
Nome: ATM Maintenance Tracker
Dados vêm de uma planilha (Google Sheets)

📌 App já possui:

Formulário de tickets
Atribuição automática de técnico
2. Adicionar Chat App

No menu:
👉 Chat apps → Create

✔ Cria integração automática com Google Chat
✔ Gera interface tipo “bot”

🎨 3. Customizar o app
🔹 Mensagem inicial

Exemplo:

Welcome to the ATM Maintenance Tracker app...

🔹 Menu (Card)
Mostra opções no chat
Ex: "My Tickets"
🔹 Slash command

Criado:

/myissues

👉 Função:

Lista tickets do usuário

📌 Conceito importante:

Slash commands facilitam uso no chat
🚀 4. Publicar o app
✔ Deployment check
Verifica erros/avisos
✔ Ajustes obrigatórios
Function: Maintenance
Industry: Financial Services
✔ Deploy
App vai para produção
💬 5. Testar no Google Chat
Criar espaço
Adicionar app
Instalar bot
Criar ticket

Campos:

Nome
ATM ID
Problema
Status
Usar comando
/myissues

👉 Retorna tickets do usuário

⚙️ 6. Automação
🔹 Evento
Quando: novo ticket criado
Tipo: ADD (inserção)
🔹 Ação

Enviar mensagem no chat:

You have created a new ticket.
🔹 Resultado

Toda vez que:

usuário cria ticket
👉 app responde automaticamente no chat
